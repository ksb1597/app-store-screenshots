## 프로젝트 개요

AI 코딩 에이전트(Claude Code 등)용 스킬로, iOS App Store 마케팅 스크린샷을 자동 생성하는 도구. Next.js 프로젝트를 스캐폴딩하고, 아이폰/아이패드 목업에 스크린샷을 넣어 Apple 요구 해상도(4가지)로 PNG 내보내기.

## 기술 스택

- Next.js, TypeScript, Tailwind CSS, html-to-image, React
- Node.js 18+ 필요
- 패키지 매니저: bun > pnpm > yarn > npm (우선순위)
